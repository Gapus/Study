# 클래스 템플릿

> 컴파일타임에 템플릿인수를 이용하여 필요한 만큼 템플릿소스를 생성합니다.

```C
template <class T, int i> 
class TempClass   
{  
public:  
    TempClass( void );  
    ~TempClass( void );  
    int MemberSet( T a, int b );  
private:  
    T Tarray[i];  
    int arraysize;  
};  
```

# 함수 템플릿

```C
template< class T > void MySwap( T& a, T& b ) {  
   T c(a);   
   a = b;   
   b = c;  
}  
```
